# 🍦 Scoopy Doopy – Ice Cream Cafe Website

A premium, fully responsive, single-page website for **Scoopy Doopy Ice Cream Cafe**, built as a portfolio showcase by **[@siteup.in](https://instagram.com/siteup.in)**.

## 📁 File Structure

```
├── index.html      # Main HTML structure (all 6 pages)
├── style.css       # All styles, animations, dark mode
├── script.js       # Interactivity, AI quiz, menu filter
└── README.md       # This file
```

## ✨ Features

- 🎨 **6 Pages** — Home, Menu, Gallery, About, Offers, Contact
- 🤖 **AI Flavour Quiz** — Claude API recommends desserts based on mood
- 🌙 **Dark / Light Mode** toggle
- 📱 **Fully mobile responsive** with hamburger nav
- ⏱️ **Live countdown timers** on offer cards
- 🔍 **Menu search + category filter** (18+ items)
- ♾️ **Auto-scrolling testimonials** carousel
- 🖼️ **Masonry gallery** with hover effects
- 🌀 **Scroll-triggered animations** throughout
- 💬 **Floating WhatsApp + Order buttons**
- 📊 **Scroll progress bar**
- 🍦 **Animated SVG ice cream** hero illustration

## 🚀 Deploy on GitHub Pages

1. Upload all 3 files (`index.html`, `style.css`, `script.js`) to a GitHub repo
2. Go to **Settings → Pages → Source → main branch → / (root)**
3. Your site will be live at `https://yourusername.github.io/repo-name`

## 🛠️ Customise

- **Business details** — Search and replace the placeholder address, phone, and email in `index.html`
- **Menu prices** — Edit the `menuItems` array in `script.js`
- **WhatsApp number** — Replace `919999999999` with the actual number in `index.html`
- **AI Quiz** — The quiz uses the Anthropic Claude API (Sonnet 4.6). Works out of the box on Claude.ai artifacts; for standalone hosting you'll need an API key proxy.
